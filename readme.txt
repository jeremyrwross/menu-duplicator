== Description ==

Quickly duplicate any WordPress menu with a very simple user interface.

== Installation ==

1. Download the latest version of the plugin files here.
1. Upload the plugin files to the `/wp-content/plugins/menu-duplicator` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Tools -> Menu Duplicator screen to duplicate any WordPress menu

== Frequently Asked Questions ==

= How do I duplicate a menu? =

Within the WordPress dashboard go to Tools -> Menu Duplicator to access the Duplicator interface.
1. Select the menu to be duplicated
1. Create a new unique name to call this duplicated menu
